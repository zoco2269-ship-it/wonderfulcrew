# WonderfulCrew 결제 플로우 (토스 페이먼츠 연동 시 참고)

## 카드 결제 (199만원)
1. 토스 페이먼츠 카드 결제
2. 완료 즉시 서비스 활성화
3. 웰컴 메시지 자동 발송

## 현금 결제 (190만원)
1. 가상계좌 발급 (건당 600원)
2. 고객 전용 계좌로 입금
3. 입금 감지 자동
4. 서비스 즉시 활성화
5. 웰컴 메시지 자동 발송

## 웰컴 메시지 내용
"🎉 원더풀크루 올패키지 결제가 완료됐어요! 지금 바로 무제한 서비스를 즐기세요.
포함 혜택: 플랫폼 무제한 이용 / 강의 영상 무제한 / 라이브 수강권 / 1:1 코치 피드백 / AI 코칭 토큰 무제한"

## 참고
- 홈페이지에 법인계좌번호 별도 게시 불필요 — 가상계좌로만 처리
- 토스 페이먼츠 API 연동은 가입 심사 완료 후 진행 예정
- 현재는 Stripe 테스트 모드로 운영 중
